AI-Powered Content Monitoring and Writing System: Revolutionizing Article Creation! 



== Description ==
Retell.media is a content monitoring and writing platform powered by artificial intelligence. Monitor news content worldwide using key filters and generate unique content based on selected articles. 

With the ability to quickly group articles using specific keywords or queries, Retell.media boosts productivity, significantly reducing the time needed for research and content creation. It provides tools for analyzing, monitoring, and rewriting articles, allowing users to choose the language and style that best suits their needs. 

Retell.media aims to help users track current articles, speed up the article writing process, and effectively manage multiple topics, making it an invaluable tool for anyone looking to optimize their content creation workflow.

Key features:

- Find content. 
Define keywords, specify news sources, and receive a flow of matching articles within minutes

- Rewrite content. 
Adopt, change styles, translate to a different languages - in a few clicks.

- Create digest.
Summarize multiple articles at once, creating a new story

- Generate articles. 
Generate articles from scratch, using prompt and quotes

Usage:

- Authenticate in the public section.

- Navigate to the WordPress Plugin section.

- Configure the "Time" field. Article request interval setting. Specify the time in minutes after which you want the plugin to request articles

- Configure the “Articles” field. Number of articles requested setting. Enter the number of articles you want the plugin to request each time.

- API Key Generation. Click the "Generate" button to have the system generate a unique API Key.

- Copy API Key. After generating the API Key, copy it using the corresponding button.

- Paste API Key. Go to the administrative panel of your WordPress site. Find the appropriate field for inserting the API Key in the plugin settings section. Paste the copied API Key into this field and save the changes.

Support:
If you encounter any issues or have any questions, please feel free to contact us at support@retell.media


== Installation ==
- Download the ZIP archive of the plugin:
Visit retell.media plugin page in the catalog. On the plugin page, click the "Download" button and save the ZIP archive of the plugin to your computer.

- Log in to your WordPress admin panel.

- Go to the "Plugins" section: find and select "Plugins" in the admin panel sidebar menu.

- Click on "Add New":
At the top of the page, click on the "Add New" button.

- Choose "Upload Plugin":
On the add new plugin page, click on the "Upload Plugin" link at the top.

- Select the ZIP archive of the "retell.media" plugin from your computer and click "Install Now"

- After installing the plugin, click "Activate".
